简单排序

Insert_sort

插入排序类比于打扑克牌，当抽牌时，玩家从牌库中抽取扑克牌，并按照大小顺序插入手牌，机制是维护有序序列，不断插入元素。

  • 模型细节：
    创建一个数组模拟手牌，初始数组为牌库。从初始数组中拿取元素（抽牌），与创建的数组中元素对比大小，找到位置（找牌），插入元素（放牌）。

  • 代码细节：
    创建insert()函数来实现插入元素的操作:

    • 找到元素位置pos
    • 将原本位置的r[pos], r[pos + 1], r[pos + 2]…全部后移
    • 将抽取元素赋值给r[pos]

    对每一个初始数组的元素调用insert()函数

  • 具体代码:

     void insert(int x){
    int pos;
    for(pos = 0; pos < tot; pos++ )
        if(r[pos] >= x) break;
    for(int k = tot; k >= pos; k--){
    	r[k+1] = r[k];
    }
    r[pos] = x;
    tot++
}

  • 优化空间：
    在原数组中整理排序，使用被浪费掉的原始空间。
    具体代码：

    void insert(int x){
    for(int i = 0; i < n; i++){
        int k, x = a[i];
        for(k = 0; k < i && a[k] < x; k++); //找到插入点k
        for(int p = i - 1; p >= k; p--){
            a[p + 1] = a[p];
        	a[k] = x;
        }
    }
}

    插入排序的复杂度为O(n^2)

Bubble_sort

冒泡排序类似于泡泡浮出水面的过程，气泡浮出水面 ，由于水压 $/rho$ gh不断减小，气泡不断变大，冒泡排序每次考虑相邻的两个元素，比较大小并将较大的放在右侧，重复n次的过程就像气泡冒出水面。
本质是每次将最大的数字放到最右侧。

  • 代码实现：
    void bubble_sort(int a[], int n){
    for(int i = 0; i < n; i++)
        for(int j = 0; j < n - 1; j++)     
        //改成j < n - i - 1可以节省每一次尾部已经排完的元素比较次数
            if(a[j] > a[j + 1]) swap(a[j], a[j + 1]);
            //swap()在algorithm库中
}

冒泡排序的复杂度也为O(n^2)

Select_sort

选择排序的原理是进行多次扫描，将每一次扫描得到的最小元素置于最做左侧，经过n次扫描可得到从小到大的一次序列

void selct_sort(int a[], int n){
	for(int i = 0; i < n; i++){
    	int cur_min = i;
    	for(int j = i; j < n; i++){
    		if(a[j] < a[cur_min]) cur_min = j;
    		swap(a[i], a[cur_min])
		}
}

选择排序的复杂度为O(n^2)

优化排序

Merge_sort

归并排序基于分治的思想，简要思路如下：

  • 将初始数组a[l, r]拆分成a[l, mid], a[mid + 1, r] 两个部分
  • 调用MergeSort(l, mid), MergeSort(mid + 1, r), 将两个小部分各自排好序
  • 将两个有序数组合并成一个大的有序数组，覆盖a[l, r]
    代码实现：
    void mergesort(int a[], int l, int r){
    if(l - r == 1) return;
    int mid = (l + r) / 2;
    mergesort(a, l, mid);
    mergesort(a, mid, r);
}
    将两个数组合并的方法：
    • 从左到右生成数组可以看成从A, B里面一个一个取元素。
    • 若A已经取完，则从B中取；若B已经取完，则从A中取。
    • A, B都没取完，则比较A, B头部元素大小，选择小的那个，大小一样则任意取。
      代码实现
      int ls = l, lt = mid, rs = mid, rt = r, pos = l;
    while(ls < lt && rs < rt){
        if(a[ls] < a[rs])
            t[pos++] = a[ls++];
        else
            t[pos++] = a[rs++];
    }
    while(ls < lt){
        t[pos++] = a[ls++];
    }
    while(rs < rt){
        t[pos++] = a[rs++];
    }
    for(int i = l; i < r; i++){
		a[i] = t[i]
    }

Quick_sort

快速排序，假设有一群人，按照身高排序，可以按照以下方式：

  • 随便选一个人

  • 比之矮的排左边，比之高的排右边

  • 把他左右边的人分别排序
    代码实现：

    void quick_sort(int a[], int l, int r){
	if(r - l <= 1) return;
    int flag = a[l];//哨兵，可以伪随机化为a[rand() % (r - l) + l， 防止被卡]
    int x = l, y = r;
    int x, y;
    for(int i = l; i < r; i++){
        if(a[i] < flag)
            t[x++] = a[i];
        if(a[i] > flag)
            t[--y] = a[i];
    }
    for(int i = l; i < x; i++)
        a[i] = t[i];
    for(int i = y; i < r, i++)
        a[i] = t[i];
    for(int i = x; i < y; i++)
        a[i] = flag;
    quick_sort(a, l, x);
    quick_sort(a, y, r);
}

  • 排序STL sort可以直接使用algorithm库排序，指定cmp函数反向排序，或者重载小于号定义
